# CUSTOMER_RELATION_PROFILE_V

**Schema:** `CCM` | **Type:** `View`

## Description
This view provides a comprehensive profile of customer relations, aggregating various mobile and fixed telecommunication service counts, bonus flags, and status information by joining core customer relation factual data with customer mapping details.
