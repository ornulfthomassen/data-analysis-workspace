# MOB_S

**Schema:** `CCM` | **Type:** `View`

## Description
This view, `CRM_ANALYSE.MOB_S`, consolidates extensive subscription data from `CCDW.SUBSCRIPTION` with detailed product information from `CRM_ANALYSE.PD` and product type configurations from two instances of `CLM_CCM.CCM_PRODUCT_TYPE_CONFIG`. It provides a comprehensive dataset for mobile subscriptions, enriched with product attributes like names, descriptions, categories, payment types, fees, and more, for analytical purposes. The data is ordered by subscription ID, start date, and end date.
