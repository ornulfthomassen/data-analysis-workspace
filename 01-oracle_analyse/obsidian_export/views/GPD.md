# GPD

**Schema:** `CCM` | **Type:** `View`

## Description
The view `CRM_ANALYSE.GPD` provides a direct, untransformed projection of all columns from the `GALAXY.PRODUCT_DIM` table or view. Its primary purpose is to expose the product dimension data to the `CRM_ANALYSE` schema under a potentially more convenient name, without applying any filtering, aggregation, or complex transformations.
