# GPD

**Schema:** `CCM` | **Type:** `View`

## Description
The view `CRM_ANALYSE.GPD` serves as a direct exposure of all columns from the `GALAXY.PRODUCT_DIM` table, effectively creating an alias or a logical layer for the underlying product dimension data without any transformations, filtering, or joins. It re-exposes the complete `PRODUCT_DIM` table under a new schema and view name.
