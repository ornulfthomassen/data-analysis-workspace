# GCP_BUDGET_TARGETS_MOBILE_DAY

**Schema:** `CCM` | **Type:** `View`

## Description
This view provides daily budget targets for various mobile-related Key Performance Indicators (KPIs). It aggregates targets for different types of sales (new, gross, upsale, swap), terminations, porting, device insurance, and specific insurance products, categorized by year, date, and multiple event channels. A key transformation is applied to standardize the 'WEB' event channel to 'Digitale Kanaler' (Digital Channels).
