# GCP_BUDGET_TARGETS_MOBILE_DAY

**Schema:** `CCM` | **Type:** `View`

## Description
This view provides mobile budget target data, primarily by selecting all columns from the 'BUDGET_TARGETS_MOBILE_DAY' source, with one transformation: it renames 'WEB' values in the 'EVENT_CHANNEL_2' column to 'Digitale Kanaler'. It effectively presents a slightly modified version of the underlying budget target data.
