# VYA_KAS_MDU_ARPU

**Schema:** `CCM` | **Type:** `View`

## Description
Provides a view of Multi-Dwelling Unit (MDU) Average Revenue Per User (ARPU) data, directly projecting columns from the KAS.MDU_ARPU table, with a calculated CUSTOMER_SK identifier.
