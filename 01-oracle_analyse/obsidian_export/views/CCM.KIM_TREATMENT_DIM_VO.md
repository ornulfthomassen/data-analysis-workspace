# KIM_TREATMENT_DIM_VO

**Schema:** `CCM` | **Type:** `View`

## Description
Creates a detailed and enriched dimension view for marketing or customer 'treatments', handling null values, standardizing data types and lengths for several attributes, and categorizing treatments into specific groups (TREATMENT_SUB_GROUP, TREATMENT_GROUP, TREATMENT_GROUP_SH) based on ACTION_CATEGORY and TREATMENT_CD.
