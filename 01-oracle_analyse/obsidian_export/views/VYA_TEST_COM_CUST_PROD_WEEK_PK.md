# VYA_TEST_COM_CUST_PROD_WEEK_PK

**Schema:** `CCM` | **Type:** `View`

## Description
This view provides a comprehensive set of aggregated metrics related to customer-product subscriptions on a weekly basis. It includes opening and closing balances, net changes for subscriptions, product subscriptions, and 'binded' subscriptions. It also captures details like period week, product, subscription type, customer segment, run/sequence IDs, number of channels, parent customer segments, and various keys related to subscription ownership, usage, and payment. The view essentially serves as a reporting layer for weekly aggregated subscription data.
