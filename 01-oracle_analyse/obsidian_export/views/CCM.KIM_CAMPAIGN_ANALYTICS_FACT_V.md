# KIM_CAMPAIGN_ANALYTICS_FACT_V

**Schema:** `CCM` | **Type:** `View`

## Description
Aggregates and transforms campaign-related detailed fact data to calculate various Key Performance Indicators (KPIs) like sales, success, presented, accepted, selected, newsales, and antichurn. It enriches the data with product, response, campaign, and treatment dimensions for analytical purposes, deriving additional categorical attributes such as sales matrix groups, customer age groups, and profit segments.
