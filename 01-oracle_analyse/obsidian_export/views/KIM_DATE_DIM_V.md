# KIM_DATE_DIM_V

**Schema:** `CCM` | **Type:** `View`

## Description
Creates a comprehensive date dimension view, including various date granularities (day, week, month, quarter, year) and several relative date calculations (e.g., current week/month/year, last 4 weeks, year-to-date, relative week/month to system date). It serves as an enriched date lookup for analytical purposes.
