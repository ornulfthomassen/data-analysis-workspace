# TV_CONTENT_PRODUCT_V

**Schema:** `CCM` | **Type:** `View`

## Description
Consolidates and standardizes information about various TV content products (including individual channels, channel packages, and basic packages) from multiple staging and QlikView data sources. It provides a comprehensive view of content products with details like product numbers, content IDs, names, segments, groups, vendors, package groups, validity periods, and source systems, effectively creating a unified catalog of TV-related content.
