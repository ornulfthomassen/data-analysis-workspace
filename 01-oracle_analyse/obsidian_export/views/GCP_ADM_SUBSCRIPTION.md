# GCP_ADM_SUBSCRIPTION

**Schema:** `CCM` | **Type:** `View`

## Description
The view GCP_ADM_SUBSCRIPTION consolidates and transforms various subscription-related data points, including customer information, device details, usage statistics (data, SMS, MMS, voice), revenue figures, and porting events, for monthly snapshots, preparing it for loading into a data analytics platform (Mjøsa/GCP).
