# SUBS_S

**Schema:** `CCM` | **Type:** `View`

## Description
Consolidates and enriches detailed subscription information by joining the primary subscription data with product dimensions and product type configurations. It provides a comprehensive view of subscriptions, including product names from different systems and descriptive attributes, for analytical purposes.
