# KIM_CCDW_SUBSCRIPTION_V

**Schema:** `CCM` | **Type:** `View`

## Description
Exposes all columns from the `ccdw.subscription` table, effectively acting as a direct wrapper or alias for subscription data within the `CRM_ANALYSE` schema. It provides a consistent interface to subscription details.
