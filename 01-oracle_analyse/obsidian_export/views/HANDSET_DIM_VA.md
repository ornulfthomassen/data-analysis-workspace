# HANDSET_DIM_VA

**Schema:** `CCM` | **Type:** `View`

## Description
Creates a Handset Dimension view by selecting all columns from 'GALAXY.HANDSET_DIM_V', ensuring non-null string representation for most columns by casting them to VARCHAR2 and replacing any NULL values with -1.
