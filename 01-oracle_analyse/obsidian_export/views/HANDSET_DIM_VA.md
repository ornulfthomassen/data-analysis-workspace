# HANDSET_DIM_VA

**Schema:** `CCM` | **Type:** `View`

## Description
Creates a dimension view for handset data, standardizing and cleansing various handset attributes by replacing null values with '-1' and casting all attributes to VARCHAR2 with specified lengths. This prepares comprehensive handset specification data for analytical or reporting purposes.
