# VYA_ORDER_LINE_MOB_HIVE_2019

**Schema:** `CCM` | **Type:** `View`

## Description
This view, named `VYA_ORDER_LINE_MOB_HIVE_2019`, consolidates and enriches detailed mobile order line transaction data. It joins a base order line fact table with various dimension tables to provide comprehensive information including order categories, termination reasons, customer demographics (age, location), subscription details, and time-related attributes. It also calculates numerous Key Performance Indicators (KPIs) related to sales, porting, product changes, terminations, and device swaps. The view specifically filters for orders with an `ORDER_STATUS_DT_KEY` before the year 2020, suggesting its purpose is to provide historical order line data, likely for the year 2019 and earlier.
