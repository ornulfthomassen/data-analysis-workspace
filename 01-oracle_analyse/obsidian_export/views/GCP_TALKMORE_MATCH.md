# GCP_TALKMORE_MATCH

**Schema:** `CCM` | **Type:** `View`

## Description
This view, "GCP_TALKMORE_MATCH", provides a comprehensive projection of customer and matching-related data sourced directly from the `TALKMORE.TALKMORE_MATCH` table or view. It applies minor data cleansing and formatting, specifically trimming leading/trailing spaces from `CUSTOMER_ID` and `ACCOUNT_ID`, and padding the `ZIP` code to a minimum length of 4 with leading zeros if necessary.
