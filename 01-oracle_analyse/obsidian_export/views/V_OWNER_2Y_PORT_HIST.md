# V_OWNER_2Y_PORT_HIST

**Schema:** `CCM` | **Type:** `View`

## Description
This view provides a historical summary of mobile subscription porting activities for each owner over the last two years (730 days). It aggregates the total number of port-in and port-out events, counts distinct main numbers, and calculates the number of active subscriptions per owner within this period. It also captures the minimum and maximum dates for both port-in and port-out events, and lists the service providers involved in these porting activities for each owner.
