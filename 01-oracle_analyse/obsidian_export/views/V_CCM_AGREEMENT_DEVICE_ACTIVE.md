# V_CCM_AGREEMENT_DEVICE_ACTIVE

**Schema:** `CCM` | **Type:** `View`

## Description
This view provides a comprehensive dataset for CRM analysis, specifically focusing on active customer agreements that are associated with devices. It exposes detailed information about the agreements (e.g., offer name, status, pricing, validity), device characteristics (e.g., IMEI, manufacturer, type, usage dates), insurance discounts, and related owner/user IDs and agreement counts. Essentially, it serves as an analytical layer for active device agreements.
