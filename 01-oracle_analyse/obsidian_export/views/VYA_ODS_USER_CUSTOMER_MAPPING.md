# VYA_ODS_USER_CUSTOMER_MAPPING

**Schema:** `CCM` | **Type:** `View`

## Description
Provides a filtered and consolidated mapping between users and customers, including registration/change dates and a link quality indicator, focusing on primary and active connections. It essentially gives a view of 'active primary user-customer relationships'.
