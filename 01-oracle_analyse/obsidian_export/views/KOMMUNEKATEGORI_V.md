# KOMMUNEKATEGORI_V

**Schema:** `CCM` | **Type:** `View`

## Description
This view provides a categorized hierarchical breakdown of municipalities. It maps municipality codes to their category, associated county (with code and uppercase name), sub-region (with code and uppercase name), and a broader national region (landsdel, with uppercase name). It primarily serves to present geographical and administrative structural data.
