# ADM_USAGE_MOBILE_CURR_V

**Schema:** `CCM` | **Type:** `View`

## Description
Aggregates mobile subscription usage and associated revenue metrics (MMS, SMS, Voice calls, and Data) for the current month. The metrics are broken down by domestic, international, roaming (EU/ROW), and specific data package categories. The data is grouped by subscription ID and period month key.
