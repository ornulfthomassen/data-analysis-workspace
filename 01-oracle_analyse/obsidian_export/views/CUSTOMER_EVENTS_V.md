# CUSTOMER_EVENTS_V

**Schema:** `CCM` | **Type:** `View`

## Description
This view provides a comprehensive snapshot of customer events, including event details, associated activities, campaigns, various KPIs, and financial metrics, specifically filtered for a single historical date: August 1st, 2018. It consolidates a wide array of event-related data for analytical purposes on that specific day.
