# CUSTOMER_EVENTS_V

**Schema:** `CCM` | **Type:** `View`

## Description
Creates a view named CUSTOMER_EVENTS_V in the CRM_ANALYSE schema. This view provides a filtered selection of customer event data, specifically for events that occurred on August 1, 2018, by extracting all listed columns from the CLM_KIM.CUSTOMER_EVENTS table.
