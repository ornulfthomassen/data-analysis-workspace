# TV_POINT_SUBSCR_V

**Schema:** `CCM` | **Type:** `View`

## Description
This view serves as a direct pass-through wrapper for the `CCM.TV_POINT_SUBSCR` object. It exposes all columns from the underlying source, providing access to subscription-related data, including credit points, used points, and tracking of changes over periods for TV and broadband services.
