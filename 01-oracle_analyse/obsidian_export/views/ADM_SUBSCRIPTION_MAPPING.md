# ADM_SUBSCRIPTION_MAPPING

**Schema:** `CCM` | **Type:** `View`

## Description
Provides a comprehensive and standardized view of historical subscription records, including various ID types (current, previous, original, past), derived date keys, and porting information, acting as a mapping layer for analytical consumption.
