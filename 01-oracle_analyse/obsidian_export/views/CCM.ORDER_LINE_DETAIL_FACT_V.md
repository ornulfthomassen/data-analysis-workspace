# ORDER_LINE_DETAIL_FACT_V

**Schema:** `CCM` | **Type:** `View`

## Description
This view provides a comprehensive, unified fact table for CRM analysis, focusing on order line details. It integrates transaction data from `ORDER_LINE_DETAIL_FACT_MV` with dimensional attributes from various sources (products, customers, dates, dealers, binding types, service providers, and customer segments). The view calculates and derives numerous Key Performance Indicators (KPIs) and descriptive attributes related to sales, product changes, porting, terminations, and customer demographics, enabling detailed reporting and analytical insights into order line performance and customer behavior.
