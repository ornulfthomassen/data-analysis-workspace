# GCP_CVC_ORDER_DETAIL

**Schema:** `CCM` | **Type:** `View`

## Description
The view `GCP_CVC_ORDER_DETAIL` serves as a comprehensive collection of detailed order information, likely for customer value chain (CVC) analysis or reporting. It selects a vast array of attributes related to orders, products (current, subscription, and previous), agreements, subscriptions, customer demographics, sales channels, events (dates, statuses, types), devices, and various Key Performance Indicators (KPIs) such as sales, churn, porting, and product changes. Essentially, it acts as an interface to a highly granular mobile order detail dataset, making all these attributes easily accessible.
