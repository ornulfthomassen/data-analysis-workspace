# VYA_KAS_PRODUKT

**Schema:** `CCM` | **Type:** `View`

## Description
This view, `VYA_KAS_PRODUKT`, serves as a direct, unfiltered interface to all product-related data stored in the `KAS.PRODUKT` table. It exposes a comprehensive set of product attributes, acting primarily as a passthrough to make product information available within the CCM schema, potentially for schema separation or to provide a specific, versioned view of the data.
