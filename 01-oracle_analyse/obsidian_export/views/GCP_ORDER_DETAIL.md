# GCP_ORDER_DETAIL

**Schema:** `CCM` | **Type:** `View`

## Description
This view consolidates a comprehensive set of detailed order, product, subscription, customer, channel, and event-related data, along with numerous key performance indicators (KPIs) such as new sales, porting, product changes, terminations, upsells, and downsells. Its primary purpose is to provide a wide, pre-processed dataset for analytical and reporting needs related to mobile order details, sales events, and various operational metrics, filtered for data where the period month key is greater than 202200.
