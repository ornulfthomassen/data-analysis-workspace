# GCP_ORDER_DETAIL

**Schema:** `CCM` | **Type:** `View`

## Description
This view creates a filtered projection of the `GCP_MOBILE_ORDER_DETAIL` table, applying explicit data type conversions to most columns. It exposes mobile order detail records where the `PERIOD_MONTH_KEY` is greater than 202200.
