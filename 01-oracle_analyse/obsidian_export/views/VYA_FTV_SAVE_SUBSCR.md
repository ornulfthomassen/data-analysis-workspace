# VYA_FTV_SAVE_SUBSCR

**Schema:** `CCM` | **Type:** `View`

## Description
Consolidates and enriches 'save' or 'churn' case data with customer, address, date, and subscription key information, providing a comprehensive view of churn prevention activities and related customer/subscription details.
