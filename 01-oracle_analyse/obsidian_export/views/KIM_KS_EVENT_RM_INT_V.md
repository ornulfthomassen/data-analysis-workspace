# KIM_KS_EVENT_RM_INT_V

**Schema:** `CCM` | **Type:** `View`

## Description
This view aggregates and transforms detailed customer interaction and event data from the 'KIM_KS_RM_INTERACTION2_V' source view, providing a comprehensive record of events with various attributes like event name, description, category, timestamps, channel information, product details, customer and user identifiers, KPI metrics, device information, and numerous custom duration and count metrics, likely for analytical purposes related to contact center operations or customer journey analysis.
