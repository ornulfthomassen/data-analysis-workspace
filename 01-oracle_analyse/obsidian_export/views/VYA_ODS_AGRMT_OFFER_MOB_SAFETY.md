# VYA_ODS_AGRMT_OFFER_MOB_SAFETY

**Schema:** `CCM` | **Type:** `View`

## Description
This view is designed to consolidate and present agreement offer data related to mobile safety, enriching it with customer mapping information. It serves as a data preparation step, likely for loading agreement reward data into an 'Mjøsa' system or for analytical purposes. It provides detailed agreement, product, customer, and dealer attributes.
