# KIM_PRODUCT_TYPE_CONFIG_V

**Schema:** `CCM` | **Type:** `View`

## Description
This view provides a standardized interface to product type configuration data by selecting all columns from the `CCM_PRODUCT_TYPE_CONFIG` table and renaming its `ID` column to `PRODUCT_KEY`. It essentially exposes the raw product type configuration details, likely for use in CRM analysis or other downstream processes, without performing complex transformations.
