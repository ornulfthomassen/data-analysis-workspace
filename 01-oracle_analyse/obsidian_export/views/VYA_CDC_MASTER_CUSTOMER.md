# VYA_CDC_MASTER_CUSTOMER

**Schema:** `CCM` | **Type:** `View`

## Description
Provides a consolidated view of master customer data by joining the `CDC.MASTER_CUSTOMER` table with `CLM_ADM.ADM_CUSTOMER_MAPPING`, filtering for specific customer mapping statuses ('EKSKUNDE', 'KUNDE').
