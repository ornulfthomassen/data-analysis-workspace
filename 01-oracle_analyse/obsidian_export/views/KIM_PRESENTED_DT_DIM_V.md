# KIM_PRESENTED_DT_DIM_V

**Schema:** `CCM` | **Type:** `View`

## Description
This view serves as a comprehensive date dimension, presenting various date and time attributes (day, week, month, quarter, year, and relative periods like current/last/next week/month/year, year-to-date, last 12 months, etc.) for analytical reporting purposes. It essentially exposes a pre-calculated set of date-related information.
