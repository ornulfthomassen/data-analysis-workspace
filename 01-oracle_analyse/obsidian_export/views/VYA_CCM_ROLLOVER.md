# VYA_CCM_ROLLOVER

**Schema:** `CCM` | **Type:** `View`

## Description
Prepares and extracts subscription rollover data from the CCM_ROLLOVER table, casting key fields to character types, for use in other systems (e.g., Mjøsa).
