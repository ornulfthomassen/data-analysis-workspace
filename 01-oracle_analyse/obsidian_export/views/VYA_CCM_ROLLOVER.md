# VYA_CCM_ROLLOVER

**Schema:** `CCM` | **Type:** `View`

## Description
Extracts rollover data related to subscriptions, including historical rollover metrics (amounts, grants, and descriptions) for the last three periods. The primary purpose is to load this 'ROLLOVER-data' into another system referred to as 'Mjøsa'.
