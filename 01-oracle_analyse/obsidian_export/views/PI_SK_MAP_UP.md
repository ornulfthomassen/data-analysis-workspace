# PI_SK_MAP_UP

**Schema:** `CCM` | **Type:** `View`

## Description
This view maps 'main_number_sk' to 'subscription_sk' and includes 'main_number_rank', providing unique combinations of these identifiers. Its primary purpose is to establish and expose the relationship between main numbers and subscriptions.
