# V_CON_SNA_AGG_MAXMONTH

**Schema:** `CCM` | **Type:** `View`

## Description
This view retrieves aggregated Social Network Analysis (SNA) metrics for the most recent month available in the source data. It selects all columns for records corresponding to the maximum 'PERIOD_MONTH_KEY'.
