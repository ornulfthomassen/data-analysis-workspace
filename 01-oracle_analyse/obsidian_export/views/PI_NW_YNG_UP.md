# PI_NW_YNG_UP

**Schema:** `CCM` | **Type:** `View`

## Description
Provides a quick overview/presentation of sales for new 'young' subscriptions, counting the occurrences for specific product offers based on various criteria like date, parameter ID, category, and a predefined list of product offer IDs.
