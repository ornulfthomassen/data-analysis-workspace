# GCP_ADM_HOUSEHOLD_SUBS_CNT

**Schema:** `CCM` | **Type:** `View`

## Description
This view, GCP_ADM_HOUSEHOLD_SUBS_CNT, acts as a passthrough or alias, exposing all columns from the underlying CLM_ADM.ADM_HOUSEHOLD_SUBS_CNT object. It provides data related to household subscription counts and associated metrics, likely for different service types (e.g., MPP, MPR, MBB, FIX, DSL, FBR) and subscription types (personal and business), along with activity duration.
