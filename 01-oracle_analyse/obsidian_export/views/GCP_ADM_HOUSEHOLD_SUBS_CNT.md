# GCP_ADM_HOUSEHOLD_SUBS_CNT

**Schema:** `CCM` | **Type:** `View`

## Description
Provides a consolidated view of household subscription counts and active days for various services (Mobile Postpaid, Mobile Prepaid, Mobile Broadband, Fixed, DSL, Fiber), differentiating between residential and business users. It primarily re-exposes data from an underlying administration table.
