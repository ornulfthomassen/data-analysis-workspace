# DEALER_DIM_VA

**Schema:** `CCM` | **Type:** `View`

## Description
The view `CRM_ANALYSE.DEALER_DIM_VA` provides a standardized and cleansed dealer dimension for analytical purposes, including sales channel categorization, by selecting and transforming data from the `GALAXY.DEALER_DIM` table. It handles null values by replacing them with a placeholder and casts columns to specific VARCHAR2 lengths.
