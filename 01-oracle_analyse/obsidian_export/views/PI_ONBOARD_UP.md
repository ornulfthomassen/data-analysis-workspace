# PI_ONBOARD_UP

**Schema:** `CCM` | **Type:** `View`

## Description
This view, created by Per Fey on 23.03.2023, is designed for uploading data for onboarding analysis. It consolidates a wide array of communication, campaign, event, response, product, user, and order-related metrics.
