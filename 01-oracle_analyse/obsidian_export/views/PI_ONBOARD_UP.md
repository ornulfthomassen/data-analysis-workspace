# PI_ONBOARD_UP

**Schema:** `CCM` | **Type:** `View`

## Description
This view is designed for uploading data specifically for onboarding analysis. It selects a comprehensive set of columns related to communication, campaigns, events, user responses, product information, user demographics, and order details, likely to support tracking and analysis of customer onboarding processes.
