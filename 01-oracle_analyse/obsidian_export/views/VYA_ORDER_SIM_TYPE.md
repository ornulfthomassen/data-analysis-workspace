# VYA_ORDER_SIM_TYPE

**Schema:** `CCM` | **Type:** `View`

## Description
Filters the 'SERVICE_ORDER_PRODUCT_PARAM' table to retrieve order parameters for a specific parameter ID (100), exposing the order ID, parameter value, and parameter status ID.
