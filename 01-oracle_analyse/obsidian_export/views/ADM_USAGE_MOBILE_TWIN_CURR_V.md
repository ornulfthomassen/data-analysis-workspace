# ADM_USAGE_MOBILE_TWIN_CURR_V

**Schema:** `CCM` | **Type:** `View`

## Description
Provides aggregated mobile usage data (MMS, SMS, Voice, Data) and associated net revenue for TWIN and DATATWIN subscriptions for the current month. It sums up various usage and revenue metrics grouped by period month, subscription ID, and sub-number.
