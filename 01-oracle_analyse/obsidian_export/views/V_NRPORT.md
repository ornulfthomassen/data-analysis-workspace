# V_NRPORT

**Schema:** `CCM` | **Type:** `View`

## Description
Provides a direct view into number portability records. It acts as an alias or a schema-isolated projection of the `NRPORT.NRPORT_PORTERINGER` table, exposing all its columns for use within the `CRM_ANALYSE` schema.
