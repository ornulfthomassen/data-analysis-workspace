# V_ANALYTICAL_MASTER_TABLE_HH

**Schema:** `CCM` | **Type:** `View`

## Description
Creates a monthly analytical master table providing a comprehensive overview of household demographics, subscription details, revenue, and service coverage information by joining various historical household and subscription data sources. It compiles a wide array of metrics related to mobile, fixed-line, and broadband services, user counts, age distribution, communication preferences, and physical address characteristics for analytical purposes.
