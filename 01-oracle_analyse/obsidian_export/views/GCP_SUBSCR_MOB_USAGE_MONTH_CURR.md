# GCP_SUBSCR_MOB_USAGE_MONTH_CURR

**Schema:** `CCM` | **Type:** `View`

## Description
Aggregates and summarizes current month mobile usage metrics, including net revenue, voice minutes, and various categories of mobile data consumption (domestic, roaming, included, package, shared bucket, reduced speed, FBB_GARANTI), for each mobile subscription.
