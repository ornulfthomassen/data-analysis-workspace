# VYA_ADM_TWIN_USAGE_MONTH_AGG_D

**Schema:** `CCM` | **Type:** `View`

## Description
Aggregates monthly mobile usage and revenue data per subscription, providing detailed breakdowns by service type (MMS, SMS, voice/TALE, data), geographical context (domestic, international, roaming), and various data plan classifications. It calculates quantities (e.g., count of MMS, minutes of voice, megabytes of data) and associated net revenues for each category.
