# VYA_HW_PR_TR_GTIN_OVRD_V

**Schema:** `CCM` | **Type:** `View`

## Description
Provides a standardized view of hardware price tier and GTIN override data, performing necessary type conversions and column renaming from its source table. It prepares GTIN, RRP (Recommended Retail Price) values (excluding and including VAT), tier, validity periods, source, manufacturer, model, color, total size, and audit information for override records.
