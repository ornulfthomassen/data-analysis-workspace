# VYA_ORDER_LINE_DETAIL_FACT

**Schema:** `CCM` | **Type:** `View`

## Description
This view consolidates detailed order line information by joining a primary order line fact table with various dimension tables, enriching the data with product attributes, customer demographics, subscription details, and derived Key Performance Indicators (KPIs) related to sales, product changes, and terminations. Its purpose is to provide a comprehensive fact view for order analysis, likely for a data warehouse or reporting system.
