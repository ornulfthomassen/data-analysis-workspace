# ADM_USAGE_MOBILE_MONTH_V

**Schema:** `CCM` | **Type:** `View`

## Description
Aggregates monthly mobile usage and revenue metrics for individual subscriptions, categorized by service type (MMS, SMS, Voice, Data) and region (domestic, international, roaming). It summarizes these metrics for the current month and the preceding five months, providing a historical overview of subscription-specific usage patterns and associated revenue.
