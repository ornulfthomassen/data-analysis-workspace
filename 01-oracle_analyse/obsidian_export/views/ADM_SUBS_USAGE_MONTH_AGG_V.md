# ADM_SUBS_USAGE_MONTH_AGG_V

**Schema:** `CLM_ADM` | **Type:** `View`

## Description
This view aggregates monthly usage data for subscriptions, combining data from main subscriptions and their 'twin' subscriptions to present a unified view. It is used for loading revenue data into the 'Mjøsa' system and for SAS IMM and BI (CCM) reporting.
