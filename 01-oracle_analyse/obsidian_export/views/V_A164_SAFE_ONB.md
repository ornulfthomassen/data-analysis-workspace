# V_A164_SAFE_ONB

**Schema:** `CCM` | **Type:** `View`

Retrieves the most recent campaign and activity details (program, campaign, activity description, dialogue ID, interaction code) for each customer, specifically filtering for 'A164' activities from January 2020 onwards.
