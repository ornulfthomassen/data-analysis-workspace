# KIM_ORDER_HANDSET_DIM_V

**Schema:** `CCM` | **Type:** `View`

## Description
Provides a dimension view for handsets associated with orders, by selecting all columns from the `KIM_HANDSET_DIM_V` and aliasing the `HANDSET_KEY` to `ORDER_HANDSET_KEY` for contextual clarity within an order-related data model.
