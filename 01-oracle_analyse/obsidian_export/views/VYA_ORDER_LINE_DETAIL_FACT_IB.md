# VYA_ORDER_LINE_DETAIL_FACT_IB

**Schema:** `CCM` | **Type:** `View`

## Description
This view provides a comprehensive set of order line details by selecting all columns from the `VYAMN_ORDER_LINE_DETAIL_FACT` table. It filters these order lines to include only those associated with the 'MOBILE' business area and an 'ÅPEN' (open) order status, by joining with the `ORDER_STATUS_DIM_MV` table.
