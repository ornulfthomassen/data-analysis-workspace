# KIM_CAMPAIGN_DIM_V

**Schema:** `CCM` | **Type:** `View`

## Description
Creates a dimension view for campaign data, standardizing various campaign attributes such as identifiers, names, descriptions, statuses, dates, and ownership information for analytical purposes. It involves casting several string columns to specific VARCHAR2 lengths.
