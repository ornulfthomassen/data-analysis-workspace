# ORDER_LINE_DETAIL_FACT_MOB_MV

**Schema:** `CCM` | **Type:** `View`

## Description
Provides a comprehensive, denormalized view of detailed order line information, encompassing various dimensions such as dates, products, pricing, sales channels, customer demographics, service providers, and key performance indicators (KPIs) related to sales, churn, and swaps. It serves as a fact table for analytical and reporting purposes, likely for mobile-related order analysis.
