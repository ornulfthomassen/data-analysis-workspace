# VYA_TM_REPORTING

**Schema:** `CCM` | **Type:** `View`

## Description
Provides a comprehensive view of Telemarketing (TM) campaign data, including customer account details, call events, agent information, disposition status, and expiry updates, specifically designed for reporting in SAS Viya.
