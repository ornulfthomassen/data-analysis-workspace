# ADM_CURRENT_SUBS_PROFIT_CAT

**Schema:** `CCM` | **Type:** `View`

## Description
This view retrieves the latest available monthly data for subscription profit categories. It provides detailed subscription information, including unique identifiers, market area, payment type, product brand, start date, active days within the period, adjusted net revenue, and two types of derived profit categories ('TALE_PROFIT_CAT' and 'PAYTYPE_PROFIT_CAT') along with their cumulative distribution values.
