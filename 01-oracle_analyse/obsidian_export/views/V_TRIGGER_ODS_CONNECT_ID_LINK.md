# V_TRIGGER_ODS_CONNECT_ID_LINK

**Schema:** `CLM_ADM` | **Type:** `View`

## Description
Retrieves the details of the most recent run (within the last 2 days) for a specific 'Connect_Id_Link' workflow and task instance ('wf_Connect_Id_Link' and 's_m_Connect_Id_Link') associated with the 'ODS' subject area. This is typically used for monitoring the status and details of a particular ETL or integration process.
