# TREATMENT_DEFAULT_V

**Schema:** `CCM` | **Type:** `View`

## Description
Creates a static lookup table providing predefined, hardcoded default values for various 'treatment' statuses or categories, such as 'Ukjent' (Unknown), 'N/A' (Not Applicable), 'Ingen treatment' (No treatment), 'Empty', and 'Treatment'. It acts as a source for generic default entries.
