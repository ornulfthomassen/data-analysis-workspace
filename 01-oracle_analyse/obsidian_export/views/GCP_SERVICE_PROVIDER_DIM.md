# GCP_SERVICE_PROVIDER_DIM

**Schema:** `CCM` | **Type:** `View`

## Description
Creates a dimension table for service providers, standardizing and cleaning various attributes such as service provider name, group, network status, and effective dates. It's intended for data warehousing purposes, specifically for loading order-related data.
