# ADM_CLM_SEGMENT_CUSTOMER

**Schema:** `CCM` | **Type:** `View`

## Description
Prepares customer segmentation data by selecting records from an existing customer segment table, applying data type conversions, deriving a period-end date and a composite key, and filtering the data to include only periods from December 2015 onwards. It presents key customer attributes like age and life phase alongside period identifiers.
