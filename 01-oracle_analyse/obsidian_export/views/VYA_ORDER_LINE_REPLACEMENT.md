# VYA_ORDER_LINE_REPLACEMENT

**Schema:** `CCM` | **Type:** `View`

## Description
This view denormalizes order line details from the `CCM.VYAMN_ORDER_LINE_DETAIL_FACT` table by joining it with numerous dimension tables. It consolidates a wide array of order-specific attributes, including product information, date attributes (for order date, status date, regret dates), dealer information, order categories, service providers, order line types, order statuses, status reasons, device details, and sales matrix data. The view includes extensive Key Performance Indicators (KPIs) and details about 'regret' (cancelled or returned) orders, making it a comprehensive dataset suitable for advanced reporting and analytical purposes related to order line performance and characteristics.
