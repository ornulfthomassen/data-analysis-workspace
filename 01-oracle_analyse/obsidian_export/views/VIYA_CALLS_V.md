# VIYA_CALLS_V

**Schema:** `CCM` | **Type:** `View`

## Description
Calculates aggregated interaction counts by year-month, client, queue type, technical result, inbound transfer group, and a derived category ('KATEGORI'). The 'KATEGORI' classifies queue types into groups like 'Faktura', 'Teknisk', 'Forhandlerservice', 'Bestilling', 'Chat', or 'Noe annet'. The view filters data to include interactions from calendar year 2022 onwards.
