# CHURN_ADM_AGRMT_FAMILIEBONUS_DET_V

**Schema:** `CLM_ADM` | **Type:** `View`

## Description
This view, 'CHURN_ADM_AGRMT_FAMILIEBONUS_DET_V', retrieves detailed agreement and 'Familiebonus' (family bonus) data from the source view 'VYA_ADM_AGRMT_FAMILIEBONUS_DET'. It filters the data to include only records where the 'PERIOD_MONTH' is from the last 25 months relative to the current date, likely for churn analysis or reporting purposes.
