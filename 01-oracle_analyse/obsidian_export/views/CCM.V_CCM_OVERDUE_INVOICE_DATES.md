# V_CCM_OVERDUE_INVOICE_DATES

**Schema:** `CCM` | **Type:** `View`

## Description
Returns dates for evaluation of Overdue Invoices, calculating start and stop dates based on historical data, and identifying today's date, type of day (e.g., holiday), and day of the week.
