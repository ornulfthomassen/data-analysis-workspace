# KIM_CONTACT_HANDSET_DIM_V

**Schema:** `CCM` | **Type:** `View`

## Description
Creates a dimension view for contact-related handset information by projecting all columns from 'KIM_HANDSET_DIM_V' and renaming 'HANDSET_KEY' to 'CONTACT_HANDSET_KEY'.
