# V_ANALYSE_USE_AGG_MAINNUMBER

**Schema:** `CCM` | **Type:** `View`

## Description
Aggregates detailed customer usage metrics (mobile calls, SMS, MMS, data volume, fixed line calls, and revenue) from a raw usage table. The data is summarized monthly, by main phone number, user customer, and business area, providing a consolidated view of usage patterns for analytical purposes.
