# GCP_ADM_CUSTOMER_OWNER_SUBS_AGG

**Schema:** `CCM` | **Type:** `View`

## Description
This view serves as a direct pass-through, exposing pre-aggregated customer owner subscription data from the 'CLM_ADM.ADM_CUSTOMER_OWNER_SUBS_AGG' source. It makes all columns from the source available under a new view name and schema ('CCM.GCP_ADM_CUSTOMER_OWNER_SUBS_AGG') without any additional transformations, aggregations, or filtering.
