# V_TRIGGER_OCS_FREE_UNITS_DAILY

**Schema:** `CLM_ADM` | **Type:** `View`

## Description
This view monitors and reports on the daily execution status and details of a specific data processing task named 'Trigger_File_to_SAS' within the 'wf_Postswap_OCS' workflow, under the 'OCS' subject area. It retrieves information like start/end times, error codes, error messages, and run status for tasks that have started today.
