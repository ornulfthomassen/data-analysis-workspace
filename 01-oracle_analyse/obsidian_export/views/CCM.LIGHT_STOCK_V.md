# LIGHT_STOCK_V

**Schema:** `CCM` | **Type:** `View`

## Description
This view, `LIGHT_STOCK_V`, serves as a direct projection of the `LIGHT_STOCK_MV` materialized view (or view). It provides a comprehensive, untransformed dataset containing various metrics related to product stock, subscriber information, usage, pricing, geographical data, and historical trends for CRM analysis. Essentially, it exposes all columns from its source view without any modifications or filtering.
