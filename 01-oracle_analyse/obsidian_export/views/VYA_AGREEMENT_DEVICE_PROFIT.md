# VYA_AGREEMENT_DEVICE_PROFIT

**Schema:** `CCM` | **Type:** `View`

## Description
This view consolidates profit-related metrics for device agreements, distinguishing between 'swap' and 'device insurance' agreements. It calculates profitability duration, COGS (Cost of Goods Sold), and other financial figures by joining detailed agreement data with existing profit records, and derives COGS sourcing logic based on available data granularity (manufacturer, model, monthly price).
