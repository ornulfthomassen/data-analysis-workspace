# KIM_CAMPAIGN_DETAIL_FACT_V

**Schema:** `CCM` | **Type:** `View`

## Description
This view consolidates detailed campaign interaction and order data from the `KIM_CAMPAIGN_DETAIL_FACT` table with various descriptive attributes from numerous dimension tables. It calculates multiple Key Performance Indicators (KPIs) such as sales (upsale, newsale, retention), presented activities, accepted responses, selected responses, anti-churn, and swaps, based on a complex set of business rules and joins. The view provides a comprehensive dataset for analyzing campaign performance, customer responses, and sales outcomes.
