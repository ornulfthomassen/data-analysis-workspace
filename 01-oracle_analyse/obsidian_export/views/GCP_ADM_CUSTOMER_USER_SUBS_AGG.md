# GCP_ADM_CUSTOMER_USER_SUBS_AGG

**Schema:** `CCM` | **Type:** `View`

## Description
Creates a view that directly exposes aggregated customer and user subscription data, including various metrics related to service usage, fees, and subscription activity over different periods, by selecting all columns from the underlying CLM_ADM.ADM_CUSTOMER_USER_SUBS_AGG source.
