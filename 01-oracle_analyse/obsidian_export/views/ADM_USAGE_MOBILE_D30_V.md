# ADM_USAGE_MOBILE_D30_V

**Schema:** `CCM` | **Type:** `View`

## Description
Aggregates mobile usage and revenue metrics for a specific reporting month (PERIOD_MONTH_KEY = 30) by subscription. It provides a detailed breakdown of MMS, SMS, voice (talk minutes and counts), and data usage (in MB), categorizing them by domestic, international (EU and Rest of World), and roaming (EU and Rest of World) activities, along with various data package and revenue types for each subscription within that month.
