# ADM_DATAPAKKE_HISTORY_SUBS_V

**Schema:** `CCM` | **Type:** `View`

## Description
This view provides a direct representation of historical subscription data package information, mirroring the `CLM_ADM.ADM_DATAPAKKE_HISTORY_SUBS` table. It exposes subscription-related metrics such as data package amounts, purchase volumes (in MB), and number of orders, categorized by total, consumer, and business segments, for CRM analysis.
