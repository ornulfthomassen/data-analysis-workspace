# SNOWDEPT_V

**Schema:** `CCM` | **Type:** `View`

## Description
Combines snow depth data with detailed geographical information from a postal code dimension table, linking them by county and municipality. The view enriches snow depth records with postal number, latitude, longitude, postal place, county code, county name, municipality code, and municipality name.
