# V_SAFE_ID_ACITVE

**Schema:** `CCM` | **Type:** `View`

## Description
Aggregates customer activity related to ID protection services, specifically tracking the activation status (active flag) and earliest activation date for email verification, credit card monitoring, and VPN token creation, grouped by customer.
