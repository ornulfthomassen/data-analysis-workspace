# ADM_PROFIT_CAT_SUBSCRIPTION

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
The procedure calculates and categorizes subscription profitability for a given period (defaulting to the previous month). It aggregates fee, usage, and roaming data, then joins it with subscription details to calculate adjusted net revenue. Finally, it computes cumulative distribution and assigns profit categories (based on percentiles) to each subscription, inserting the results into a partitioned historical table.
