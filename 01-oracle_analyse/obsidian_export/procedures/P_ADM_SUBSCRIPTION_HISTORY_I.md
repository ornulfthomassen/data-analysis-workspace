# P_ADM_SUBSCRIPTION_HISTORY_I

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
Populates or updates a monthly partition of the `ADM_SUBSCRIPTION_HISTORY_I` table by joining subscription history data with current customer mapping information for a specified month (`P_YYYYMM`). It performs a source data availability check, creates a temporary table with the processed data, and then uses a partition exchange operation to load the data into the main partitioned table.
