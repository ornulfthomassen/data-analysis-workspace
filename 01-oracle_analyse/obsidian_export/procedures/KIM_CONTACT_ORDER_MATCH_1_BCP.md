# KIM_CONTACT_ORDER_MATCH_1_BCP

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
This procedure identifies and matches customer campaign details with corresponding order information. It populates two temporary tables by filtering and transforming data from various fact and dimension tables. It then iterates through aggregated order details derived from these temporary tables and other dimensions, and updates the main 'CRM_ANALYSE.KIM_CAMPAIGN_DETAIL_FACT' table with matched order attributes (e.g., order ID, order dates, product keys, dealer information, binding benefit descriptions). The procedure also logs its execution status and record counts.
