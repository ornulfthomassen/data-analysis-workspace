# TCX_TEAM_PROJECTS_TRG

**Schema:** `CCM` | **Type:** `Procedure`

## Description
Automatically assigns a unique project ID to new rows being inserted into the `TCX_TEAM_PROJECTS` table by fetching the next value from the `TCX_TEAM_PROJECTS_SEQ` sequence.
