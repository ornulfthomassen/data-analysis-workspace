# FORMAT_CLM_LIVSFASE_5C

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
Maps a numeric input representing a life phase segment ID to a standardized Norwegian life phase description (e.g., 'Familie', 'Ung voksen', 'Senior'). It handles NULL input and returns 'Ukjent', and returns 'ERROR' for unmapped numeric inputs.
