# STOR_FORBOKSTAV

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
This is a scalar function named `STOR_FORBOKSTAV` (Swedish for 'capital first letter'). It takes a single string as input, converts its first character to uppercase, and all subsequent characters to lowercase. The function then returns this newly formatted string. It effectively converts a string to 'proper case' or 'title case' for a single word.
