# PP_GCP_MOBILE_ORDER_DETAIL

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
This procedure processes mobile order line details for a given month range. It enriches the order data with churn information by joining it with a churn mapping table. The results are stored in a temporary table, which is then used to exchange a partition (or create one if it doesn't exist) in the permanent target table. This effectively loads or refreshes monthly partitions of mobile order detail data.
