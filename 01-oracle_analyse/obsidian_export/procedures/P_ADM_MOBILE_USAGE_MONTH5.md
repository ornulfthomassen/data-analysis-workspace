# P_ADM_MOBILE_USAGE_MONTH5

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
Calculates and aggregates monthly mobile usage statistics (MMS, SMS, Voice, Data, and associated revenues) for subscriptions. It then creates a temporary table with these statistics for a specified month and uses partition exchange to load this data into the `ADM_MOBILE_USAGE_MONTH_NEW3` table, managing the creation and replacement of monthly partitions.
