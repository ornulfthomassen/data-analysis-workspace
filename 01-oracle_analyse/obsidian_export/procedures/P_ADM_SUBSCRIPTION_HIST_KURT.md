# P_ADM_SUBSCRIPTION_HIST_KURT

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
This procedure constructs a historical snapshot of subscription data for the previous month (derived from `SYSDATE`). It first validates the availability and consistency of source data. Then, it creates two temporary tables (`TMP1_SUBSCRIPTION_HIST_KURT`, `TMP2_SUBSCRIPTION_HIST_KURT`) by joining various dimension and fact tables from the `GALAXY`, `CCDW`, and `CLM_ADM` schemas. It includes data cleansing logic for the first temporary table. Finally, it uses data from these temporary tables to build the main `ADM_SUBSCRIPTION_HIST_KURT` table. The existing `ADM_SUBSCRIPTION_HIST_KURT` table is backed up by renaming it to `ADM_SUBSCRIPTION_HIST_KURT_OLD` before the new one is created. The procedure also logs its execution status and potential errors to an external history table.
