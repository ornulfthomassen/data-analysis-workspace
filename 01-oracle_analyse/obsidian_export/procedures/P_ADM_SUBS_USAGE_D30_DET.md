# P_ADM_SUBS_USAGE_D30_DET

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
Aggregates subscription usage data for a 30-day period and loads it into a partitioned permanent analytical table, replacing an existing partition. It handles temporary table creation, partition management, and statistics gathering.
