# P_ADM_CUST_CHAIN_AFFILIATION

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
Calculates and aggregates customer-dealer chain affiliation metrics for a specified month, including first/last order details and various sales KPIs. This aggregated data is then loaded into a monthly partitioned table, managing partition creation and handling existing data conflicts.
