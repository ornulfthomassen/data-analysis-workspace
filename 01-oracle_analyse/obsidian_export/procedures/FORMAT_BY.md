# FORMAT_BY

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
Translates a single-character code representing a type of settlement (e.g., 'B' for 'By', 'T' for 'Tettbygd') into its corresponding descriptive string. It returns 'Ukjent' for NULL input, and 'ERROR' for unmapped inputs.
