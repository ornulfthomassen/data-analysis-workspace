# P_ADM_SUBS_USAGE_MOB_MONTH_AG2

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
Calculates and aggregates mobile subscription usage data (MMS, SMS, Voice, Data) for a specified month. It categorizes usage by roaming and destination zones, and traffic types, then stores the aggregated results as a partition in a permanent administrative table.
