# STOCK_MOBILE_HIST2

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
Processes mobile subscription data by extracting raw details, enriching with date dimensions, and aggregating subscription counts and hardcoded discount values into a daily historical snapshot. All intermediate and final tables are created temporarily and refreshed daily.
