# P_CU_ODS_CUSTOMER_PROF_ACCESS_AGG

**Schema:** `CCM` | **Type:** `Procedure`

## Description
Aggregates customer profile information, including subscription and agreement counts by product group (e.g., mobile, broadband, TV) from both user and owner perspectives, as well as device agreements and insurance details. It also includes Talkmore specific subscription counts. The processed data is loaded into a new temporary table, which then replaces the existing permanent aggregate table via a rename (swap) operation, provided the change in row count is within an acceptable threshold. The procedure also handles indexing and privilege grants for the new table.
