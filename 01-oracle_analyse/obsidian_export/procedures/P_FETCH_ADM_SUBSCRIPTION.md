# P_FETCH_ADM_SUBSCRIPTION

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
This procedure extracts subscription data for the current and previous months from source tables, performs transformations (casting, trimming, aliasing), and loads this processed data into new, month-specific historical/staging tables. It dynamically names the target tables based on the month and includes logic to drop existing tables and log operation details and potential errors.
