# P_ADM_CUSTOMER_USAGE

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
This procedure calculates monthly customer usage metrics for a specified month (P_YYYYMM), differentiating between owner and user subscriptions. It aggregates detailed subscription data from core customer and subscription aggregation tables, then stores the results into a new partition of the ADM_CUSTOMER_USAGE table. It utilizes a temporary table for intermediate data staging and performs a partition exchange for efficient and atomic data loading into the target table.
