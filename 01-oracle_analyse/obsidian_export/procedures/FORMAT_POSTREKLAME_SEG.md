# FORMAT_POSTREKLAME_SEG

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
Translates a short input code (P_IN) representing a 'POSTREKLAME_SEG' (Post advertising segment) into a descriptive Norwegian string. It acts as a lookup/mapping function, returning 'Ukjent' for NULL input, a specific description for known codes, and 'ERROR' for unknown codes.
