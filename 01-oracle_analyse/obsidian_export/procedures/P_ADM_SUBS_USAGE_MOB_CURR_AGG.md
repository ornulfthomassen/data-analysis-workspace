# P_ADM_SUBS_USAGE_MOB_CURR_AGG

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
This procedure aggregates mobile subscription usage data for a specific month (defaulting to 999912, representing December 9999, as per the script's default initialization for V_YYYYMM). It first processes detailed usage records by enriching them with various KPI categorizations (roaming zone, destination zone, traffic type, data package type, shared bucket type) from several dimension tables into a temporary detailed table. Subsequently, it aggregates this detailed data by period, subscription ID, and sub-number, calculating sums of events, durations, volumes, and net revenue across numerous categories for MMS, SMS, Voice, and Data usage, further broken down by roaming/destination zones and data package types. Finally, it uses this aggregated data to replace a partition in the permanent `ADM_SUBS_USAGE_MOB_MONTH_AGG` table.
