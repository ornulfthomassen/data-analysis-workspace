# P_ADM_MOBILE_USAGE_MONTH

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
The procedure calculates monthly aggregated mobile usage statistics (including MMS, SMS, Voice calls, Data usage, and associated revenues) per subscription. It constructs a temporary table with these aggregates, handles the creation of the corresponding monthly partition in the main administrative table if it doesn't exist, and then efficiently loads the data from the temporary table into the specific partition of the `ADM_MOBILE_USAGE_MONTH` table using an `ALTER TABLE ... EXCHANGE PARTITION` operation.
