# FORMAT_CLM_LIVSFASE_6C

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
Maps a numeric 'life phase' segment ID to a descriptive Norwegian string based on a predefined `CASE` statement logic, re-categorizing some original segments (e.g., 'Barn' and 'Ungdom' are mapped to 'Småbarnsfamilie' and 'Etablert barnefamilie' respectively, while others are mapped directly or to a combined category).
