# P_CCM_TALKMORE_KURTMAPPING

**Schema:** `CCM` | **Type:** `Procedure`

## Description
This procedure performs a multi-stage matching process to link 'Talkmore' user data with existing customer records from various 'Galaxy' and 'CDC' data sources, and external 'REFERENCE.CONSUMER_CUST_INFO' data. It starts by extracting and aggregating Talkmore user and owner details into a temporary table. Then, it sequentially attempts to match these Talkmore records with customer dimension data and historical addresses using increasingly less strict criteria (exact name/address/birthdate, historical address, partial name matches, postcode only, and finally external contact info). It populates a primary 'TALKMORE_MATCH' table with the best available match, recording the type of match found. Multiple temporary tables are used as intermediate steps during this process.
