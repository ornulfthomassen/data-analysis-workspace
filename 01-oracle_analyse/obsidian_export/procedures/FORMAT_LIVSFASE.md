# FORMAT_LIVSFASE

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
Translates specific input codes representing life stages (livsfase) into more descriptive Norwegian phrases. It maps predefined codes like 'EnsligForsørger' to 'Enslig forsørger', handles NULL input as 'Ukjent', and returns 'ERROR' for unmapped inputs.
