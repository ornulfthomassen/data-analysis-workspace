# P_ADM_MOBILE_USAGE_MONTH3

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
This procedure calculates and aggregates mobile usage data (MMS, SMS, Voice, and Data volumes and revenues), categorizing it by service type, domestic/international/roaming status, and various product/traffic dimensions for a specified month. The aggregated data is then loaded into a monthly partition of a permanent analytical table (`CLM_ADM.ADM_MOBILE_USAGE_MONTH_NEW`). It manages the creation of temporary tables and partitions as part of this process.
