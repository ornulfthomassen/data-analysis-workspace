# KIM_CAMPAIGN_ORDER_MATCH_HIST

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
This procedure processes historical order line and campaign data to establish campaign attribution for order lines. It first stages relevant order line details from `GALAXY.ORDER_LINE_DETAIL_FACT_MV` and related dimensions, then stages campaign contact and treatment details from `CRM_ANALYSE.KIM_CAMPAIGN_DETAIL_FACT` and its dimensions. It then joins these staged data sets based on various matching criteria (customer, product, date, etc.) into a combined temporary table. Finally, it applies complex matching, ranking, and attribution logic to determine which campaign influenced each order line and inserts the result into the permanent history table `CRM_ANALYSE.ORDER_LINE_CAMPAIGN_PRIVATE_H`.
