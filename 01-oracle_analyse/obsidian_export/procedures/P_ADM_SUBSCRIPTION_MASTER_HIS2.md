# P_ADM_SUBSCRIPTION_MASTER_HIS2

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
This procedure processes and transforms subscription data for various service types (mobile, fixed telephony, fixed broadband). It stages raw subscription data, calculates core subscription metrics and relationships, and handles port-in/port-out events. Finally, it consolidates mobile subscription data into a comprehensive historical master table, enriching it with product and dealer information, and historical porting details. The process involves creating and updating multiple intermediate and final data layers, with backup mechanisms for key tables.
