# P_CREATE_LIGHT_STOCK

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
The procedure rebuilds two tables. First, it drops and recreates `CRM_ANALYSE.CCM_PRODUCT_SUBSCRIPTION` as a copy of `CLM_CCM.CCM_PRODUCT_SUBSCRIPTION`, and then adds indexes to it. Second, it drops and recreates `CRM_ANALYSE.LIGHT_STOCK_T`. This table is populated with denormalized, aggregated data derived from a large number of CRM, product, customer, subscription, terminal, and usage-related source tables, combining various metrics and attributes into a single comprehensive view.
