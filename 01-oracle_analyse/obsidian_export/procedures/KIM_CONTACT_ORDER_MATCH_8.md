# KIM_CONTACT_ORDER_MATCH_8

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
This procedure identifies and matches relevant order details to campaign contact records. It first stages a filtered subset of campaign details into a temporary table, then uses complex product-matching logic based on household IDs and product attributes to find the 'best' order for each campaign contact. Finally, it updates the main campaign detail fact table with the matched order's key information and logs the process status.
