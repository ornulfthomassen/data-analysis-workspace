# DBMS_NUMSYSTEM

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
This Oracle PL/SQL package, `DBMS_NUMSYSTEM`, provides a set of utility functions for converting numbers between different numeral systems, including binary, decimal, octal, and hexadecimal representations. It contains functions to convert from binary, octal, and hexadecimal to decimal, and vice-versa.
