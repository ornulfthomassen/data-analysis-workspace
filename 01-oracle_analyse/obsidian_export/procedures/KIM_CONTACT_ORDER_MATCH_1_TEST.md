# KIM_CONTACT_ORDER_MATCH_1_TEST

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
Matches campaign details with order line items based on product, main number, and customer keys. It aggregates relevant order information and uses it to update specific order-related attributes in the `KIM_CAMPAIGN_DETAIL_FACT` table. The procedure also logs execution statistics.
