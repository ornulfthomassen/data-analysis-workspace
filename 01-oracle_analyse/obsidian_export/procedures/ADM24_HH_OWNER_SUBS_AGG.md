# ADM24_HH_OWNER_SUBS_AGG

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
This procedure creates and/or populates the 'ADM_HH_OWNER_SUBS_AGG' table, which aggregates household-level subscription, revenue, and traffic data for specified months. It checks for the existence of the target table and its partitions, creating them if necessary, and then inserts aggregated data from various source tables. The aggregation includes counts of different subscription types, maximum/minimum active days, and summed traffic/revenue metrics per household per month.
