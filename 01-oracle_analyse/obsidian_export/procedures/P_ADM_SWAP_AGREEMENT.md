# P_ADM_SWAP_AGREEMENT

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
The procedure processes agreement and subscription swap data for a specified month (`P_YYYYMM`). It constructs a temporary table by joining and transforming data from various source systems, including agreement details, product information, subscription history, and IMEI data. After populating this temporary table, it uses an 'exchange partition' strategy to efficiently load the processed data into a dedicated monthly partition of the permanent `CLM_ADM.ADM_SWAP_AGREEMENT` table. It also handles the creation of the partition if it doesn't already exist and performs statistics collection.
