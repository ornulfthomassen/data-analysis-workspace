# RESPONSE_RANK

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
Compares the 'RESPONSE_RANK' of two provided 'RESPONSE_KEY' values from the 'KIM_RESPONSE_DIM' table. It returns 1 if the rank of the first key is greater than the rank of the second key, otherwise, it returns 0.
