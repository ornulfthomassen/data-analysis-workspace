# P_ADM_CUSTOMER_USE_USER

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
Calculates and updates monthly customer usage and subscription aggregated data for a specified month (`P_YYYYMM`). It stages the aggregated data from various customer and subscription sources into a temporary table, then uses a partition exchange operation to update the corresponding monthly partition in the `ADM_CUSTOMER_USE_USER` permanent table.
