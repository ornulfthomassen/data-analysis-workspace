# P_ADM_SUBSCRIPTION_CORE_FIXBB

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
The procedure processes raw fixed broadband subscription data, transforming and aggregating it into an intermediate temporary core table. It then uses this temporary data to incrementally load (insert new records and update existing ones) a permanent fixed broadband subscription core table, handling historical and current subscription states, and enriching with product and dealer dimensions. It creates backup tables before modifying the main tables.
