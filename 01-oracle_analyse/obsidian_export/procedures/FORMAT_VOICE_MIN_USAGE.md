# FORMAT_VOICE_MIN_USAGE

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
Categorizes a numeric input, representing voice minute usage, into predefined textual ranges (e.g., 'Nullbruk', '1 - 60 min', '61 - 120 min', 'Over 120 min'). It handles null or invalid negative inputs by returning 'ERROR'.
