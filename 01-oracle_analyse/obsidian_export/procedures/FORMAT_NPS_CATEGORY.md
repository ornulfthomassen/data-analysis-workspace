# FORMAT_NPS_CATEGORY

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
Categorizes a numeric Net Promoter Score (NPS) likelihood-to-recommend value into 'Promoter', 'Passive', or 'Detractor' based on predefined score ranges (0-6 Detractor, 7-8 Passive, 9-10 Promoter). It returns an empty string for NULL input and 'ERROR' for out-of-range values.
