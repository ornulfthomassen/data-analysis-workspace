# FORMAT_AGE_SEGMENT

**Schema:** `CCM` | **Type:** `Procedure`

## Description
This function takes a numeric input (P_IN) and categorizes it into one of four predefined segments (0, 1, 2, or 3) based on its value. It returns the segment number as a numeric type. The `TRIM` function applied to the numeric return value has no functional effect in this context.
