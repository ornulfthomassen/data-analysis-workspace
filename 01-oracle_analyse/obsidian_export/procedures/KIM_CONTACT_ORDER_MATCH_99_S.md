# KIM_CONTACT_ORDER_MATCH_99_S

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
Identifies and updates previously unmatched order records in the CRM_ANALYSE.KIM_CAMPAIGN_DETAIL_FACT table, setting their ORDER_MATCH_KEY to 99 and populating associated order details. It also logs the procedure's execution status into CLM_CCM.GOV_DQ_MARTS and updates internal logging statistics in STLOG.ST_IN.
