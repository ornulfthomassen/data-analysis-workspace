# PP_GCP_TMP_MOBILE_ORDER_LINE

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
The procedure creates or recreates a detailed mobile order line table (`CLM_ADM.TMP_MOBILE_ORDER_LINE`) by joining data from a central order line fact table with multiple dimension tables. It filters the data based on `PERIOD_MONTH_KEY` and `ORDER_SOURCE` parameters and then gathers statistics on the newly created table.
