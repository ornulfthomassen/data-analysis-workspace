# P_ADM_SUBS_USAGE_CURR_DET

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
This procedure aggregates daily subscription usage data into a temporary table based on various keys and sums of usage metrics. It then uses Oracle's partition exchange mechanism to efficiently load this aggregated monthly data into a specific partition of a permanent, partitioned monthly summary table. The process includes dropping existing temporary tables, creating a new partition if needed, populating the temporary table, altering column properties (NOT NULL), exchanging the temporary table's data into the permanent table's partition, and gathering statistics for the newly loaded partition.
