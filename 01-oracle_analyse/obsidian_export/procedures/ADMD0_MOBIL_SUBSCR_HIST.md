# ADMD0_MOBIL_SUBSCR_HIST

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
This procedure constructs a detailed mobile subscription history by first creating several temporary staging tables (`ADM_MOBIL_SUBSCR_HIST_RAW`, `ADM_SUBSCR_IN_PORT`, `ADM_SUBSCR_OUT_PORT`, `ADM_DEALER_START`) from various source systems, and then joining these temporary tables and other dimension tables to populate the final `ADM_MOBIL_SUBSCR_HIST` table. It includes logic for identifying subscription changes, porting events, and dealer information.
