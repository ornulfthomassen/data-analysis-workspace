# STOCK_MOBILE_HIST_MONTH

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
Manages and populates monthly partitioned history tables ('STOCK_MOBILE_HIST_MONTH_DET' for detailed data and 'STOCK_MOBILE_HIST_MONTH_AGG' for aggregated data). For a given month range, it dynamically creates these tables if they don't exist, drops and re-adds partitions for each month, and then loads data into these partitions by performing an `ALTER TABLE ... EXCHANGE PARTITION` operation with source tables (`STOCK_MOBILE_HIST_DET` and `STOCK_MOBILE_HIST_AGG`) that are presumably populated by an external procedure (`CLM_ADM.STOCK_MOBILE_HIST`). It also handles index creation, statistics gathering, and granting select privileges.
