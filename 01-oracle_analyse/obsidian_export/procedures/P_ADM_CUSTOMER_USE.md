# P_ADM_CUSTOMER_USE

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
Calculates and stores customer usage and social network analysis (SNA) metrics for a specified month. It populates a temporary table with these metrics derived from multiple source tables, creates a new partition in the permanent target table (if it doesn't exist), and then atomically replaces the partition's data with the contents of the temporary table.
