# P_CCM_TALKMORE_CUSTOMER_MAPPING

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
This procedure performs multi-stage customer matching. It identifies Talkmore subscribers (from TALKMORE.INVENTORY_USERS) and attempts to link them to existing customer records in the GALAXY data warehouse (GALAXY.CUSTOMER_DIM) using a hierarchy of matching criteria. Criteria range from full matches on name, date of birth, and address (current or historical), to partial name matches, and finally to matches using external customer information sources. The process populates a 'TALKMORE_MATCH' table with the Talkmore customer's data augmented by the best-matched GALAXY customer details and a 'MATCH_ID' indicating the match confidence.
