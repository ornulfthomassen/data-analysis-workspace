# FORMAT_MAP2_6C

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
Maps a numeric input (P_IN) representing a segment ID (MAP2_6C) to a predefined descriptive string label in Norwegian. It handles specific segment IDs (1-6), NULL, and -1 by returning 'Ikke segmentert', and returns 'ERROR' for unmapped values.
