# P_ADM_SUBSCR_MASTER_HIST

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
This Oracle SQL procedure generates comprehensive historical master data for mobile and fixed-line subscriptions. It creates several intermediate staging tables to prepare the data, including raw subscription details, enriched details with historical context (previous, original subscriptions, days between subscriptions), and mobile porting (inbound/outbound) events. Finally, it consolidates this information into two main historical tables: one temporary (`ADM_SUBSCRIPTION_MASTER_HIST`) and one permanent (`TDM_SUBSCRIPTION_MASTER_HIST`), calculating change reasons and various derived attributes for detailed historical analysis. The procedure supports selective execution for specific tables via the P_TABLE parameter.
