# ADM21_SUBSCRIPTION_AGG

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
Aggregates subscription, traffic, and fee data from various source tables into a partitioned summary table, CRM_ANALYSE.ADM_SUBSCRIPTION_AGG, for specified monthly periods. It handles the creation of the target table, its partitions, and indexes as needed, and logs the process.
