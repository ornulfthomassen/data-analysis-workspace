# P_CU_ODS_CUSTOMER_PROF_SVC_MOB

**Schema:** `CCM` | **Type:** `Procedure`

## Description
The procedure calculates and consolidates various customer profile and service-related information, including mobile subscription history/status, family bonus agreement details, and safety product usage. It stores this aggregated data into temporary work tables, then combines it with base customer information, and finally performs a table swap to update the main ODS_CUSTOMER_PROF_SVC_MOB table. This ensures an atomic update of the central customer profile service table.
