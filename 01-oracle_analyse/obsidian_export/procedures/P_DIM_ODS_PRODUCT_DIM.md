# P_DIM_ODS_PRODUCT_DIM

**Schema:** `CCM` | **Type:** `Procedure`

## Description
Performs a full load (rebuild and swap) of the ODS_PRODUCT_DIM table. It populates the table with product dimension data derived from CRM_ANALYSE.PD and various ODS subscription and agreement tables. The procedure calculates active product flags, applies data quality checks using a row count threshold, and manages the table swap with backup and index handling.
