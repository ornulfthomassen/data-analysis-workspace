# P_ADM_PROFIT_CAT_MND_SUBS_HIST

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
This procedure calculates monthly profit categories for mobile subscriptions. It aggregates subscription-related data, including period details, subscription characteristics, fees, and usage (both domestic and roaming). These aggregated metrics are then joined to compute an 'adjusted net revenue' per subscription. Finally, it uses cumulative distribution to assign overall and payment-type-specific profit categories (1-7) to each subscription, storing these results in a historical, partitioned table.
