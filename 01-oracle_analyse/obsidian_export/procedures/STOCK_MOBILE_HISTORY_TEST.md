# STOCK_MOBILE_HISTORY_TEST

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
Processes daily mobile subscription data for a given date, enriching it with customer demographics, product attributes, profit categories, and discount information. It aggregates this detailed data to produce a daily mobile history (detailed and aggregated views), including specific handling for Talkmore subscriptions and various discount types. The process involves multiple temporary tables to stage and transform the data.
