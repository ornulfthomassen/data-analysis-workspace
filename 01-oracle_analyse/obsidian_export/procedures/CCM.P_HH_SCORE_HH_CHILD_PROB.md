# P_HH_SCORE_HH_CHILD_PROB

**Schema:** `CCM` | **Type:** `Procedure`

## Description
Calculates and stores the probability of a household having children for customer segmentation purposes. It processes household, customer, address, and mobile subscription data, applies demographic and dwelling type factors, normalizes probabilities, and predicts child age segments based on statistical data. The results are loaded into the CLM_CCM.HH_CHILD_PROBABILITY table using a temporary table and swap mechanism.
