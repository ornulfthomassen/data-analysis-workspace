# FORMAT_RESIDENCE_TYPE

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
Maps a numeric building or residence type code to a descriptive string based on a predefined set of rules. It translates numerical input representing various property categories (e.g., 'Ene/tomannsbolig', 'Rekke/flermannsbolig', 'Blokk/terasse', 'Næringsbygg', 'Offentlig bygg') into a Norwegian text description.
