# ADM11_1_HANDSET_AGG

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
This procedure first aggregates handset historical data per subscription from the ADM_SUBSCR_HANDSET_HIST table into a new table named ADM_SUBSCR_HANDSET_HIST_AGG. It calculates first and last usage dates, first and last occurrence of various handset attributes (producer name, model name, OS type, etc.) for each subscription and handset model. Subsequently, it further aggregates this data by handset model into another new table named ADM_HANDSET_HIST_AGG, calculating counts and min/max values for various attributes across all subscriptions using that model. It dynamically drops existing versions of these aggregation tables before recreating them.
