# P_ORDER_LINE_DETAIL_FACT_MOB

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
The procedure dynamically creates or recreates a permanent fact table named 'ORDER_LINE_DETAIL_FACT_MOB'. This table aggregates and transforms detailed order line data by joining information from various dimension tables, customer segments, and other fact tables. It calculates various KPIs related to sales, porting, and terminations, and includes customer demographic data and product information. Additionally, it sets the parallelism for the 'CRM_ANALYSE.PROFITSEGMENT_MOBILE' table.
