# KIM_CONTACT_DETAIL_BO_AGG

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
Refreshes and aggregates campaign contact detail data into a central fact table (KIM_CAMPAIGN_REPORT_FACT). The procedure first drops existing bitmap indexes from the target table, then truncates it, and finally repopulates it by combining and aggregating data from 'KIM_CAMPAIGN_DETAIL_FACT' and 'KIM_CAMPAIGN_NO_TREATMENT' tables. After data insertion, it recreates all previously dropped bitmap indexes on the target table.
