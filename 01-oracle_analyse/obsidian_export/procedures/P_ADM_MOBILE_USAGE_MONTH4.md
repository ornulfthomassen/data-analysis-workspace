# P_ADM_MOBILE_USAGE_MONTH4

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
This procedure calculates and aggregates mobile usage statistics (MMS, SMS, Voice, and Data volumes/revenues) for a specified month. It populates a temporary table with these aggregated metrics, then performs a partition exchange to load the data into a monthly partitioned, permanent 'ADM_MOBILE_USAGE_MONTH_NEW2' table, replacing or adding the data for the given month. It also manages the creation of partitions if they don't exist and drops the temporary table.
