# P_SU_ODS_SUB_EQUIPM_DEVICE_INFO

**Schema:** `CCM` | **Type:** `Procedure`

## Description
Aggregates device, handset, and agreement information for mobile subscriptions, then updates the `ODS_SUB_EQUIPM_DEVICE_INFO` table using a table swap strategy. It creates several temporary work tables, populates a new version of the main table, and then renames it to replace the old one, providing device-related profile information for main and sub numbers.
