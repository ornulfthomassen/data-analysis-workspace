# P_HH_ODS_HOUSEHOLD_STOCK_AGG

**Schema:** `CCM` | **Type:** `Procedure`

## Description
Aggregates household mobile subscription stock and fixed-line product stock by household address categories, storing the results in a permanent household stock aggregation table. It uses a create-new-then-swap pattern to update the main table, retaining a backup of the old data.
