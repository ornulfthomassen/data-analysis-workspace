# ADM21_SUBSCRIPTION_AGG_I

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
This procedure processes subscription data for a given month-year period (or the previous month if none specified) to aggregate traffic and revenue metrics. It checks for the existence of the target aggregation table `CRM_ANALYSE.ADM_SUBSCRIPTION_AGG_I`. If the table does not exist, it creates it as a partitioned table along with local indexes. For each month in the specified range, it verifies the completeness of data in several source tables/views. If data for a month is not yet present in the aggregation table, it creates a new partition for that month (if necessary) and inserts aggregated data, including various mobile traffic (MB, MMS, SMS, Voice) and revenue figures, by joining subscription history, temporary traffic data, and galaxy fact tables.
