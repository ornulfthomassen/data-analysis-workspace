# KIM_CONTACT_ORDER_MATCH_1

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
The procedure identifies and matches campaign contacts with their most relevant order details based on complex ranking criteria, creating temporary tables for intermediate data processing. It then updates the main CRM campaign fact table (`KIM_CAMPAIGN_DETAIL_FACT`) with the matched order information and logs the execution status and results.
