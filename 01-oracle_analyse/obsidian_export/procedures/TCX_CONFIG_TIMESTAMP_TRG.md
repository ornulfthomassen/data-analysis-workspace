# TCX_CONFIG_TIMESTAMP_TRG

**Schema:** `CCM` | **Type:** `Procedure`

## Description
Automatically sets or updates the 'CONFIG_TIMESTAMP' column to the current system date and time for new or modified rows in the 'CCM.TCX_CONFIG' table.
