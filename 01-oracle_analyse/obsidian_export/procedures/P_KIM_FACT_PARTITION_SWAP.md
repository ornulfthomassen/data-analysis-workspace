# P_KIM_FACT_PARTITION_SWAP

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
Rebuilds and updates a specific partition of the `KIM_CAMPAIGN_DETAIL_FACT` table based on the provided contact month. It achieves this by creating a temporary table with data transformed from the original fact table and several mapping views, then performing a partition exchange to replace the old partition with the new data.
