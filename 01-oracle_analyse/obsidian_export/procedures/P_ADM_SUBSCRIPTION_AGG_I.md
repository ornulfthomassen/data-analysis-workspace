# P_ADM_SUBSCRIPTION_AGG_I

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
This procedure aggregates subscription usage and revenue data for a specified month (P_YYYYMM) and loads it into a partitioned summary table, ADM_SUBSCRIPTION_AGG_I. It first validates the existence and completeness of source data, creates a temporary table with the aggregated results, and then atomically exchanges this temporary table with a corresponding partition in the permanent target table. It includes robust error handling and logging mechanisms.
