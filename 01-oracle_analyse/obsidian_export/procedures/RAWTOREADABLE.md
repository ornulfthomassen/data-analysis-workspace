# RAWTOREADABLE

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
This Oracle function, named RAWTOREADABLE, takes a RAW datatype value as input and converts it to a VARCHAR datatype. The conversion happens implicitly when the RAW input is assigned to a VARCHAR2 variable. It then returns the converted VARCHAR value.
