# P_CHURN_INSIGHT_CHURN

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
This procedure analyzes customer and subscription data to generate churn insights. It aggregates detailed subscription information, customer demographics, and churn forecast metrics from various source views. It creates a temporary wide-format table, enriches it with additional customer agreement data, and finally loads the consolidated data into a permanent churn insight table, truncating any existing data in the target prior to insertion.
