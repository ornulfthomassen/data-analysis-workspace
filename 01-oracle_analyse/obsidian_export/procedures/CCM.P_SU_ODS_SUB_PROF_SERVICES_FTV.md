# P_SU_ODS_SUB_PROF_SERVICES_FTV

**Schema:** `CCM` | **Type:** `Procedure`

## Description
This procedure implements a 'swap' mechanism for the `ODS_SUB_PROF_SERVICES_FTV` table. It loads transformed and aggregated data from various source tables into a new temporary table (`ODS_SUB_PROF_SERVICES_FTV_N`). After successful data loading, index creation, and statistics gathering, it performs a conditional table swap: the existing `ODS_SUB_PROF_SERVICES_FTV` table is renamed to `ODS_SUB_PROF_SERVICES_FTV_O` (as a backup), and the temporary table `ODS_SUB_PROF_SERVICES_FTV_N` is renamed to `ODS_SUB_PROF_SERVICES_FTV`, making it the new main table. The swap operation is conditional, based on the percentage change in row counts between the old and new data, and includes error handling and status logging.
