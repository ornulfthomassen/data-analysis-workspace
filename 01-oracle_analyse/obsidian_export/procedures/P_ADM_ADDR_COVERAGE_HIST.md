# P_ADM_ADDR_COVERAGE_HIST

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
This procedure updates a monthly partitioned historical table, ADM_ADDR_COVERAGE_HIST, for the previous month's data. It first attempts to add a new partition for the previous month. It then populates a temporary table (TMP_ADDR_COVERAGE_HIST) with transformed data from CLM_CCM.ODS_COVERAGE, including deriving certain flags and casting data types. Finally, it uses an 'EXCHANGE PARTITION' operation to replace or load the data into the corresponding partition of the permanent ADM_ADDR_COVERAGE_HIST table, ensuring data consistency and integrity through checks and logging.
