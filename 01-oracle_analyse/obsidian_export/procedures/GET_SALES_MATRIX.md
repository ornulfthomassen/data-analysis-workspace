# GET_SALES_MATRIX

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
Classifies a sales scenario into a specific 'sales matrix' category (e.g., Newsale, Upsale, Downsale, Neutrale, Error, Missing) based on various input parameters such as market area, KPI product change, new sale indicators, rank comparisons, brand comparisons, and matrix type.
