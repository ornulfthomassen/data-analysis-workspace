# P_EV_ODS_EVENT_ORDER_MOB_ONL

**Schema:** `CCM` | **Type:** `Procedure`

## Description
This procedure performs a delta load of mobile order event data. It identifies new records from a source view based on the latest processed date in the target table, stages them in a temporary table, filters them according to specific business rules (e.g., non-null IMEI, product ID changes, ownership transfer, or new sales actions), and then inserts the qualified records into the permanent target table. Additionally, it purges records older than 730 days from the permanent target table and refreshes table and index statistics.
