# P_ADM_SUBSCR_TWIN_HIST

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
Populates the `ADM_SUBSCR_TWIN_HIST` partitioned table with historical data about subscription twins and associated handset details for a specific month (`P_YYYYMM`). It includes validation checks to ensure necessary source data exists, manages the creation and exchange of table partitions, and logs its execution status using an external procedure.
