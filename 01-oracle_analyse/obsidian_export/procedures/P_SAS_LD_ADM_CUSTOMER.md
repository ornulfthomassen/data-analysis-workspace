# P_SAS_LD_ADM_CUSTOMER

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
This procedure loads historical customer data from monthly-suffixed source tables (e.g., CLM_ADM.ADM_CUSTOMER_YYYYMM) into a temporary staging table (CLM_ADM.TMP_CHURN_ADM_CUSTOMER). It first truncates the target table, then iteratively inserts data for 26 months, starting from 25 months prior to the current date and moving up to the current month (or the most recent month in the 26-month window).
