# TTT

**Schema:** `CCM` | **Type:** `Procedure`

## Description
This procedure manages the existence and structure of two tables: `ODS_COVERAGE` and `ODS_COVERAGE_TMP`. It ensures that `ODS_COVERAGE` exists in the `CLM_CCM` schema with a predefined column structure. If `ODS_COVERAGE_TMP` exists, it is dropped and then recreated as an empty table with the same column structure as `ODS_COVERAGE`. The procedure primarily handles schema-level operations (CREATE/DROP TABLE) rather than data manipulation.
