# P_ADM_MONTH_DIM

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
The procedure refreshes the `CLM_ADM.ADM_MONTH_DIM` table. It first archives the existing `ADM_MONTH_DIM` table by renaming it to `ADM_MONTH_DIM_OLD` (after dropping any pre-existing `ADM_MONTH_DIM_OLD`). Then, it creates a new `CLM_ADM.ADM_MONTH_DIM` table by joining multiple instances of `CLM_ADM.ADM_MONTH_DIM_V` to derive current, next, and six previous month's dimension data. Finally, it grants `SELECT` privileges on the newly created table to `CRM_ANALYSE` and `CCM`.
