# P_ADM_SUBSCRIPTION_CORE_FIX_T

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
Maintains and processes fixed telephony subscription core data and associated porting information. It constructs a raw subscription dataset, processes it into a core analytical dataset identifying subscription transitions and historical changes, and manages 'port-in' and 'port-out' records for fixed telephony subscriptions. The procedure performs full data refreshes for staging tables and incremental updates/inserts for main analytical tables.
