# P_ADM_DEVICE_AGREEMEN2

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
This procedure manages a partitioned table, 'ADM_DEVICE_AGREEMEN2', by refreshing or creating a specific monthly partition. For a given month (determined by `P_YYYYMM` or the previous month if not provided), it checks for and drops an existing temporary table, creates a new temporary table (`TMP_ADM_DEVICE_AGREEMENT`) by selecting and joining data from `ADM_DEVICE_AGREEMENT` and `ONL_REP.SERVICE_ORDER_PRODUCT` for that specific month, and then uses a partition exchange operation to integrate this temporary data into the 'ADM_DEVICE_AGREEMEN2' table as a new or refreshed partition. It also adds a new partition to 'ADM_DEVICE_AGREEMEN2' if it doesn't exist for the target month and computes statistics on the updated partition.
