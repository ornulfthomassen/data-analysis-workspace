# P_ADM_MOBILE_USAGE_MONTH2

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
This procedure aggregates monthly mobile usage data from various fact and dimension tables. It calculates detailed metrics for MMS, SMS, Voice (TALE), and Data usage, including domestic, international, roaming, and revenue breakdowns. The aggregated results for a specified month (P_YYYYMM) are first stored in a temporary table. Subsequently, this temporary table is used to exchange data with a specific monthly partition of the permanent table 'ADM_MOBILE_USAGE_MONTH_NEW' in the 'CLM_ADM' schema, ensuring efficient data loading and partition management. It also handles partition creation if the target partition does not exist.
