# P_ADM_PROFIT_CAT_SUBS_HIST

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
The procedure calculates various subscription-level profit metrics for mobile services for a given period. It begins by validating the availability of source data for the specified period. It then creates several intermediate tables to gather period-related dimensions, extract subscription details, aggregate fee components, and sum usage metrics (both general and roaming). These intermediate results are combined to derive adjusted net revenue, which is then used to categorize subscriptions into profit segments based on cumulative distribution. Finally, the comprehensive historical data, including the calculated categories, is loaded into a partitioned history table named ADM_PROFIT_CAT_SUBS_HIST via a partition exchange operation.
