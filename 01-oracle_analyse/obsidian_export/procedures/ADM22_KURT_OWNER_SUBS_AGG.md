# ADM22_KURT_OWNER_SUBS_AGG

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
This procedure aggregates subscription and usage data by owner (KURT_ID_OWNER) for specified months. It manages the creation and partitioning of the target aggregation table (ADM_KURT_OWNER_SUBS_AGG) if it doesn't exist, and then inserts aggregated data from various CRM_ANALYSE and GALAXY source tables into this partitioned table for each month within the given range (or the previous month by default). It also handles index management (drop/recreate) and logs the load history.
