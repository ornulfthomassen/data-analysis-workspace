# P_BROADBAND_COMPETITION

**Schema:** `CLM_ADM` | **Type:** `Procedure`

## Description
The Oracle SQL procedure `P_BROADBAND_COMPETITION` calculates and consolidates various broadband competition and densification metrics across different geographical levels (address, grunnkrets, municipal, county, postcode). It integrates data from multiple source systems, processing customer subscriptions, product information, campaign offers, network infrastructure (Home Connected/Home Passed), project details, NKOM (Norwegian Communications Authority) coverage, and VULA (Virtual Unbundled Local Access) agreements, and competitor feedback. The procedure constructs a comprehensive address-level dataset, derives competition segments and reasons, and stores the enriched data into a main `BCS_BROADBAND_DENSIFICATION` table. It also maintains a historical record in `BCS_BROADBAND_DENSIFICATION_HIST` and generates a final summary table `BCS_BROADBAND_COMPETITION` with key subscription and competition segment information.
