# KIM_CONTACT_ORDER_MATCH_3

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
Matches contacts to their most relevant orders based on product group and main number, and updates a campaign detail fact table with the matched order details. It first creates a temporary filtered fact table, then processes each contact's best order, and finally logs the process status and record counts.
