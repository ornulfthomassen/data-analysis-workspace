# KIM_CONTACT_ORDER_MATCH_5

**Schema:** `CRM_ANALYSE` | **Type:** `Procedure`

## Description
Updates `CRM_ANALYSE.KIM_CAMPAIGN_DETAIL_FACT` with identified order matching details. It correlates customer campaign contacts with order line items based on product, household, and date criteria, using a temporary table for intermediate processing and logging execution status.
